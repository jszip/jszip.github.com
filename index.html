Consume JavaScript packages from Maven builds

Just add the <dependency> and go. We can even aggregate all the scripts into a single minified file for you

Use Maven to build JavaScript packages

Just set <packaging>jszip</packaging> and go. We'll even generate the package.json for you

Live development

Fail fast. Fix fast. Then just hit reload. Works across all modules within the Maven reactor

The JSZip project is exploring an alternative way of packaging JavaScript modules in order to make consuming dependent JavaScript modules easier for people developing with Apache Maven 3.
